# 更新日志

## 2026-06-20

### 歌曲信息

- 为歌曲列表增加封面，并在歌曲进入视口时加载。
- 封面使用浏览器 Cache Storage 缓存，有效期为 90 天。
- 增加水鱼公投拟合定数，并统一显示两位小数。
- 为部分特殊歌曲增加标签和简短说明。

### 筛选与抽取

- 增加“抽取歌曲”功能，可从筛选结果中随机抽取 1–4 首歌曲。
- 增加“我不想玩”标记，被标记的歌曲不会参与筛选结果和随机抽取。
- 不想玩的歌曲会持久化保存在浏览器中，并可从折叠面板恢复。

### 谱面确认

- 每首歌曲增加“谱面确认”入口。
- 电脑端打开哔哩哔哩网页搜索。
- 移动端优先尝试拉起哔哩哔哩 App，失败时可手动打开网页搜索。

### 应用体验

- 增加浅色、深色和跟随系统主题。
- 增加 PWA 支持，可安装到桌面或移动设备。
- 优化移动端布局、数据缓存和离线回退。
